# 322COM Raycaster
A multithreaded raycaster with support for complex models and Phong shading.
## Controls
### Render
- **ENTER**: Re-render
### Camera
- **W**: Forwards
- **S**: Backwards
- **A**: Left
- **D**: Right
- **R**: Up
- **F**: Down
